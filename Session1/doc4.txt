Jest framework - 
    JavaScript testing framework developed by Facebook. 
    It's designed to be simple to set up and use while providing powerful features for writing and executing tests. 
    Jest is particularly well-known for its ease of use, speed, and comprehensive functionality tailored for testing JavaScript code, especially React applications. 
   
Pros - 
    Integration with React, Vue, Angular Node, Express or prettymuch any JS framework
    Faster And Parallel execution of test
    Good Documentation and developer support are available
    Also had good features like mock, spies, matchers etc.
